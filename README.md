# yup-tail

```
NAME:
   tail - output the last part of files

USAGE:
   tail [OPTIONS] [FILE...]

      Print the last 10 lines of each FILE to standard output.
      With more than one FILE, precede each with a header giving the file name.
      With no FILE, or when FILE is -, read standard input.

COMMANDS:
   help, h  Shows a list of commands or help for one command

GLOBAL OPTIONS:
   --lines value, -n value  output the last NUM lines, instead of the last 10 (default: 10)
   --bytes value, -c value  output the last NUM bytes (default: 0)
   --follow, -f             output appended data as the file grows (default: false)
   --quiet, -q, --silent    never output headers giving file names (default: false)
   --verbose, -v            always output headers giving file names (default: false)
   --help, -h               show help
```
